# useIsomorphicEffect

一个实用的 React Hook，提供 IsomorphicEffect 相关功能。

## 功能特性

- ✅ **核心功能**: 提供 IsomorphicEffect 的核心功能
- ✅ **类型安全**: 完整的 TypeScript 类型支持
- ✅ **易于使用**: 简洁的 API 设计
- ✅ **性能优化**: 高效的实现方式

## 基本用法

```typescript
import useIsomorphicEffect from "./useIsomorphicEffect";

function Component() {
  const result = useIsomorphicEffect();
  
  return <div>使用 useIsomorphicEffect</div>;
}
```

## 兼容性

- ✅ React 16.8+
- ✅ TypeScript 4.0+
- ✅ 现代浏览器
